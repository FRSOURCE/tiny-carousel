# Plugin Scroll Snap Fallback

This module provides automatic support mechanism and polyfill for CSS scroll-snap featureopen in new window. You can force polyfill usage using force configuration property.

To learn on how to install and use plugin, please see our usage guide.

# PluginConfig

  • Properties:

    • {boolean} force

  • Details:

    Config is an object passed along to the carousel.use method. force configuration property allows you to override the result of support detection mechanism and force usage of JavaScript CSS Scroll Snap polyfill. By default value: false.
