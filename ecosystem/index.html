# Core

The main entry point is always a core package, which provides the most basic carousel API and allows extending of it functionalities.

For now, the only and only core package is a @frsource/tiny-carousel-core (opens new window). Itâ€™s a CSS scroll snap points-based carousel system. Usage of this native API makes is snappy, lightweight and accessible solution.
